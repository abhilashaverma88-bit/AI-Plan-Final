The GTM Intelligence Agent — Full Build Spec

Everything you will have built by Day 29, in plain English. Follow these steps exactly in n8n.

INPUT

5 competitor RSS / news feeds + brand positioning doc uploaded as knowledge base

TRIGGER

Automatic schedule: every Monday 7:30am via n8n Schedule Trigger node

PROCESSING

4 chained AI Agent nodes: Monitor → Synthesise → Brief → Content

OUTPUT

Email brief to inbox + 3 LinkedIn drafts in Google Doc + Telegram alert for urgent signals

MEMORY

Vector store holds 8 weeks of briefs — agent tracks trend direction over time

COST

~$3–5/month OpenAI API. n8n free tier covers this workflow comfortably.

  1. Sign up for n8n Cloud (free trial)Go to n8n.io → Start for free. Use the cloud version. No server, no install, no credit card.
  2. Add credentials: OpenAI API + Gmail OAuthSettings → Credentials → New. Add OpenAI API key (from platform.openai.com). Add Gmail via Google OAuth. Total: 10 minutes.
  3. Build the Schedule Trigger nodeNew workflow → Add node → "Schedule Trigger" → Every Monday, 7:30am. This is the heartbeat of your entire agent.
  4. Add 5 RSS Feed Read nodes (one per competitor)Add node → "RSS Feed Read" → paste competitor blog RSS URL. Repeat ×5. Connect all 5 into one Merge node. Find RSS URLs by appending /feed or /rss to any blog URL.
  5. Add AI Agent #1 — Monitor AgentConnect Merge node to AI Agent. Prompt: "You are a competitive intelligence analyst. Identify product launches, pricing changes, campaign themes, or strategic shifts. Return as JSON array: {brand, signal_type, description, urgency_score_1_to_5}."
  6. Add AI Agent #2 — Synthesis AgentPass Monitor JSON here. Prompt: "Synthesise the top 3 signals. For each: Signal Summary (2 sentences), Strategic Implication for our brand, Recommended Response we should take this week."
  7. Add AI Agent #3 — Brief Writer (RAG-enabled)Upload your brand positioning PDF to n8n vector store. Connect to this node. System prompt: paste 2 samples of your writing style. Task prompt: "Write a 400-word Monday Morning Intelligence Brief using the brand context and competitive signals provided."
  8. Add AI Agent #4 — Content AgentPrompt: "From the intelligence brief, write 3 LinkedIn posts. Post 1: data-led insight. Post 2: opinion/POV. Post 3: question for the audience. Each 150-200 words, matching the writing tone in [your examples]."
  9. Add Google Docs node — save posts for reviewContent Agent → Google Docs → Append to doc titled "Week of [date] — Content Drafts." You review and edit before posting — the agent drafts, you decide.
  10. Add Gmail node — deliver brief to your inboxBrief Writer → Gmail → To: yourself. Subject: "Monday Intelligence Brief — [date]." Body: full 400-word brief + link to Google Doc with posts.
  11. Add IF node + Telegram/WhatsApp for urgent alertsAfter Monitor Agent: IF any urgency_score >= 4 → Message node → "🔴 URGENT SIGNAL: [brand] — [description]. Full brief Monday 9am." Set up Telegram bot via @BotFather in the Telegram app (5 minutes).
